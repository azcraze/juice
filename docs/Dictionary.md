# Dictionary Help

Word, yo<br/>Parts of this cog are adapted from the PyDictionary library.

# define
 - Usage: `?define <word> `

Displays definitions of a given word.

# antonym
 - Usage: `?antonym <word> `

Displays antonyms for a given word.

# synonym
 - Usage: `?synonym <word> `

Displays synonyms for a given word.
