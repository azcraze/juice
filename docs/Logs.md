# Logs Help

Owner cog to show the latest logs.

# logs
 - Usage: `?logs <lines> `
 - Restricted to: `BOT_OWNER`

Sends the last n lines of the latest log file (default 100).

## logs private
 - Usage: `?logs private `

Whether logs should be sent to your DMs.

## logs file
 - Usage: `?logs file `

Sends the entire latest log file.
