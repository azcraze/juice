# OnlineStats Help

Information about what devices people are using to run discord.

# onlinestatus
 - Usage: `?onlinestatus `
 - Aliases: `onlinestats`
 - Checks: `server_only`

Print how many people are using each type of device.

# onlineinfo
 - Usage: `?onlineinfo [member] `
 - Checks: `server_only`

Show what devices a member is using.
