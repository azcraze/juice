# ApiTools Help

API tool to get/post data.

# apitools
 - Usage: `?apitools `
 - Restricted to: `BOT_OWNER`

Group for API tools.

## apitools post
 - Usage: `?apitools post <url> [headers_and_data] `

Send a HTTP POST request.

## apitools get
 - Usage: `?apitools get <url> [headers] `

Send a HTTP Get request.
